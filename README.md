# Gas Cylinder Weighing System (IOT)

## Overview
Designed a smart IT-Obased system for real-time gas cylinder weight monitoring using STM32 and ESP32.

## Key Contributions
- Implemented load-cell interface using HX711 ADC
- Displayed real-time weight on I2C LCD
- Transmitted data wirelessly via 433 MHz RF modules
- Migrated system to ESP32 for Wi-Fi and cloud integration
- Optimized power using deep sleep techniques

## Technologies Used
STM32, ESP32, ADC, UART, RF Communication, Wi-Fi, Blynk IoT

## Key Highlights
- Low-power embedded system design
- Wireless communication integration
- IOT-based monitoring solution
